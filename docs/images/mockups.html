Instrucciones para Exportar

Opción 1 - Capturas de pantalla:

  1. Abra este archivo en Google Chrome
  2. Use la extensión "Full Page Screen Capture" o herramienta similar
  3. Capture cada sección individualmente
  4. Guarde las imágenes en la carpeta docs/images/

Opción 2 - Exportar a PDF:

  1. Presione Ctrl+P (o Cmd+P en Mac)
  2. Seleccione "Guardar como PDF"
  3. Configure los márgenes a "Ninguno"
  4. Active "Gráficos de fondo"

Opción 3 - Herramienta de recorte:

  1. En Windows: Use Win+Shift+S
  2. Seleccione cada mockup individualmente
  3. Pegue en un editor de imágenes
  4. Guarde como PNG

Tip: Para mejor calidad, use un zoom del 100% en el navegador antes de capturar.
